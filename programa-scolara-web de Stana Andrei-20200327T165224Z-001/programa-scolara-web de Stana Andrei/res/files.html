Introducere:

Probabil că atunci când ai încercat să rezolvi probleme cu matrice, a fost destul de deranjant când trebuia să introduci o matrice mai mare. Mai mult, dacă programul tău nu afișa un răspuns corect, după ce corectai programul trebuia să introduci matricea din nou.
Sunt sigur că ai salvat imagini, documente sau fișiere pe calculator pentru a le putea folosi mai târziu. Aceste fișiere sunt deschise de diverse programe. Ar fi foarte util dacă am putea să ne salvăm matricile, vectorii sau numerele în fișiere și să îi spunem programului nostru să citeasca din fișier, pentru a nu trebui să introducem datele de fiecare dată de la tastatură.
În C++ putem face acest lucru în felul următor:



sintaxa: 
#include < fstream>
using namespace std;

int main() {
  ifstream in("data.in");
  ofstream out("data.in");
  int a,b,c,d;
  in>>a>>b>>c;
  d=a+b+c;
  out<< d;
  out.close();
  return 0;
}


Programul de mai sus citește 3 numere a,b si c din fișierul data.in și afișează suma lor în fișierul data.out; În video-ul de mai jos poți vedea cum să folosești fișierele text în CodeBlocks:









Explicatie:

"fstream" e numele antetului in care gasim tipurile de date ifstream si ofstream.
Cu ifstream putem crea o variabila pe care sa o folosim pentru a citi date din fisierul aflat intre paranteze.
Cu ofstream putem crea o variabila pe care o folosim pt. a scrie date in fisierul aflat intre paranteze
Pentru ca modificarile facute intr-un fisier dupa o operatie de scriere sa fie salvate folosim functia close();

Variabilele de tip ifstream si ofstream fuctioneaza ca si cin si cout.





CodeBlocks:

Pentru a folosi fișiere text în cadrul unui proiect în CodeBlocks, trebuie să dai click pe File > New > Empty file sau să apeși Ctrl + Shift + N, să apeși Yes la Do you want to add this new file in the active project?. Înainte să dai Save, trebuie să îi dai fișierul un nume și să alegi opțiunea pentru Save as type să fie ultima din listă, adică All files(.).
Execută pasul de mai sus pentru fiecare fișier pe care vrei să îl folosești în program.
Pentru a deschide fișierele create în CodeBlocks, apasă pe + din stânga folderului Others și dă dublu click pe fiecare fișier pe care vrei să-l deschizi, alege Open it inside the Code::Blocks editor și apasă OK.
Acum poți scrie date în fișierele create și să le citești sau să le scrii în alt fișier din main.cpp.



Citirea până la finalul fișierului

Din fericire nu trebuie să specificăm întotdeauna la începutul fișierului câte numere urmează să citim sau să marcăm cu 0 finalul datelor: 


#include < fstream>
#include < iostream>
using namespace std;

int main() {
  ifstream fin ("input.txt");
  int x;
  while(fin >> x) {
    cout << x << " ";
  }
  return 0;
}

La rulare se va afișa 3 -235 72 2 3 4.
Acest lucru este posibil deoarece fin >> x va deveni fals în momentul în care nu se mai poate citi din fișier.
